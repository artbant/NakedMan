# NUKED MAN

2D пиксельный платформер в чёрно-белом стиле с дизерингом. Наш герой — обычный человек с шляпой, который после бурной вечеринки проснулся в другом измерении. Теперь нужно пробиваться сквозь монолитные миры, разрушать стены, спасать заложников и биться с гигантскими боссами-глазами.

## Как запустить

Игра чистый vanilla JS без сборки. Но **нельзя открывать через `file://`** — Web Audio API и некоторые функции не работают.

```bash
cd путь/к/проекту
python3 -m http.server 8000
# затем в браузере: http://localhost:8000
```

Или любой другой локальный сервер (`npx serve`, `php -S`, и т.д.).

**Важно для разработки:** открой DevTools → вкладка Network → галочка "Disable cache". Иначе браузер запомнит старые `.js` файлы и новые правки не применятся.

## Управление

- **←/→** или **A/D** — движение
- **↓** или **S** — присесть (hitbox уменьшается до 16px, движение медленнее)
- **Space** или **W** или **↑** — прыжок. В воздухе можно прыгнуть ещё раз (**double jump**).
- **Z** или **X** — удар. Бьёт перед собой по 3 точкам (верх/середина/низ тела).
  - **↓ + Z** — удар вниз (ломает тайл под ногами)
  - **Прыжок + Z в воздухе** (vy < -50) — удар вверх (ломает потолок)
- **Esc** — пауза
- Тап по **иконке ноты** в правом верхнем — вкл/выкл музыку
- Тап по слайдерам **M/S** — регулировка громкости музыки и эффектов

## Структура проекта

```
index.html        — главный файл, подключает всё
constants.js      — константы (размеры мира, тайла и т.д.)
game-state.js     — объект game со всем состоянием сессии
particles.js      — система частиц (взрывы, крошка, дым)
levels.js         — ручные ASCII-уровни + парсер (см. LEVELS.md)
map.js            — процедурная генерация, тайлы, биомы, снег, портал
player.js         — игрок: спрайт, физика, удар
enemies.js        — все 6 типов врагов + снаряды + AI (см. ENEMIES.md)
items.js          — пауэрапы (аптечка, сигарета) и бочки
hostages.js       — спасаемые NPC-заложники
boss.js           — босс "Глаз" на тонких ногах
hud.js            — UI: сердца, скор, музыка, победа, титул
game.js           — главный цикл, input, адаптивная музыка
```

## Что уже сделано

### Игровой мир

- **Процедурная генерация карты** каждые запуски, ширина 130+ тайлов
- **Вся геометрия разрушаема**: монолит ломается ударами. Прочность растёт с глубиной (1 HP на поверхности, 2 HP в среднем слое, 3 HP глубже)
- **4 биома** меняются каждые 4 уровня: VOID (базовый), DUST (пыльный с туманом), GLARE (плотные звёзды), HIVE (звёзды-кластеры)
- **Параллакс неба** — звёзды едут в 10× медленнее игрока, создавая глубину
- **Снег идёт снизу вверх** (фича, подчёркивает абсурдность другого измерения)
- **Босс на каждом уровне** — гигантский глаз на тонких ногах, 3 фазы по HP

### Игрок

- **3 жизни** максимум (не 4). Аптечки редкие.
- **Double jump** — второй прыжок в воздухе даёт облачко частиц
- **Coyote time** 100 мс — можно прыгнуть 0.1с после схода с края
- **Jump buffer** 150 мс — если нажал прыжок до приземления, прыжок сработает автоматически
- **Приседание** — уменьшает hitbox до 16 px, замедляет движение, проверяет потолок при выходе
- **Sprint** — после 2 сек непрерывного движения камера смотрит дальше вперёд (lookAhead 40→100)
- **Speed boost** — подбери сигарету, 5 сек быстрого движения

### Враги (6 типов, 3-кадровая ходьба)

- **patrol** (E) 18×24 — ходит туда-сюда, 2 HP
- **guard** (G) 16×22 — стреляет с замахом 0.25с, 1 HP
- **runner** (R) 16×22 — убегает, паникует, 1 HP
- **flyer** (F) 20×14 — летает, пикирует сверху вниз, 1 HP
- **sniper** (S) 14×22 — дальнобойный стрелок с прицельным лучом 0.5с, 2 HP
- **bruiser** (U) 24×30 — ломает стены, отступает после удара, 3 HP
- Все **реагируют на звук** (разрушение тайла) — patrol разворачивается, guard становится alerted
- Все **следят глазами за игроком** — зрачки поворачиваются в твою сторону
- Все **моргают** раз в 4-6 секунд
- Подробнее — см. [ENEMIES.md](ENEMIES.md)

### Босс — "THE EYE"

- Огромный глаз 48×48 на двух тонких ногах (4 тайла длиной)
- Зрачок следит за игроком
- HP зависит от уровня (lvl 1 → 6, lvl 5 → 12, lvl 10 → 20)
- **Фаза 1** (100-50% HP): одиночные лазеры каждые 2.5 сек
- **Фаза 2** (50-20% HP): двойные лазеры + наклонный выстрел каждые 1.8 сек
- **Фаза 3** (20-0%): тройные веером + призыв runner'ов каждые 5 сек
- Портал не работает пока босс жив
- Респавн при смерти игрока — если босс был жив

### Процедурная адаптивная музыка

Рифтово-рок ритм в до-минорной пентатонике 118 BPM с **адаптивными слоями**:

- **Базовый**: бас + хэт (всегда)
- **Мелодия**: когда игрок движется
- **Sprint**: верхняя октава при sprintTimer > 1с
- **Combat**: когда враг ближе 200 px — клэп + снэр + power-chord + crash + **тремоло-дрон** и **диссонантный кластер** (в стиле Кубрика / Пендерецкого)
- **Boss active**: усиленный combat + второй басовый кластер + глиссандо каждый такт
- **Boss phase 3**: dread-слой с супернизким ревущим дроном (ROOT-36) + непрерывное нисходящее глиссандо

Плюс **стинги на события**: удар, убийство, пикап, урон, спасение, взрыв, появление босса, смена фазы, призыв приспешников, смерть босса (победный мажорный аккорд).

### HUD

- 3 сердца (только актуальные, без пустых)
- Score по центру вверху (если > 0)
- Кнопка музыки + слайдеры громкости M/S в правом верхнем
- Счётчик врагов снизу-слева
- Speed-boost бар (при активной сигарете)

### Рекорды

Сохраняются в `localStorage`:
- Лучший score (все уровни вместе)
- Лучшее время прохождения одного уровня
- Громкость музыки и эффектов

Показываются на титульном экране и экране победы, с миганием `NEW BEST TIME` если побил.

## ASCII-уровни

Вместо только процедурной генерации можно писать уровни вручную в `levels.js` в текстовом формате. Парсер читает массив строк и автоматически создаёт карту. Подробности — см. [LEVELS.md](LEVELS.md).

## Советы по отладке

- Все файлы пишутся vanilla JS без сборки — можно редактировать прямо на лету
- Порядок подключения в `index.html` важен — зависит кто от кого
- `game` в `game-state.js` — единый state object, всё состояние там
- `player` в `player.js` — объект игрока
- `enemies`, `hostages`, `items`, `objects`, `projectiles`, `ragdolls`, `particles`, `boss` — массивы/объекты в соответствующих файлах
- В DevTools Console можно прямо менять `player.lives = 99` или `game.currentLevel = 5` и тестировать

## Философия проекта

Всё рисуется через `ctx.fillRect` — никаких спрайтов-картинок. Цветовая палитра: 2 цвета (чёрный `#000` и белый `#fff`). Дизеринг Байера 4×4 даёт градации. Это сужение создаёт узнаваемый визуальный стиль и позволяет быстро итерировать.
