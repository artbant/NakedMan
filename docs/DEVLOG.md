# DEVLOG — Nuked Man

## Сессия 1 — 14 апреля 2026

### Заложили фундамент
- ✅ Концепция: голый мужик в шляпе проснулся в другой вселенной, идёт домой
- ✅ Стек: HTML5 Canvas + pure JS, всё через fillRect (никакого сглаживания)
- ✅ Персонаж нарисован попиксельно из SVG (15×24px), стоит спиной
- ✅ Базовое движение, прыжок, коллизия с тайлами
- ✅ Тайл 24×24px, 8 типов текстур
- ✅ 3-слойный параллакс фон (луна, монолиты, фоновые блоки — всё fillRect)
- ✅ Пиксельный HUD (сердца + текст через fillRect, никаких ctx.font)
- ✅ Камера скроллится только вправо
- ✅ Проект на GitHub: artbant/NakedMan
- ✅ Claude Code настроен в VS Code

### Механики
- ✅ Анимации персонажа: idle (покачивание), run (4 кадра), jump, punch (3 фазы)
- ✅ Удар кулаком: Z/X + кнопка HIT, хитбокс, частицы + вспышка + звёздочки
- ✅ Knockback при ударе — враги отлетают, игрок слегка откатывается
- ✅ Система жизней: 3 старт, макс 5 с бутылками
- ✅ Смерть → respawn на том же уровне; 0 жизней → Game Over → новый уровень

### Карта
- ✅ Процедурная генерация: длина 42–60 тайлов, случайные высоты и текстуры
- ✅ Пропасти строго 1–3 тайла (гарантированно перепрыгивается)
- ✅ 3 типа прочности тайлов: хрупкий (1 удар), средний (2 удара), прочный
- ✅ Маркеры разрушаемости + трещины при повреждении
- ✅ Удар разрушает тайлы перед персонажем
- ✅ Плавающие платформы всегда хрупкие

### Враги
- ✅ Тип A — патрульный (12×16px): ходит туда-обратно, 2 HP
- ✅ Тип Б — сторожевой (guard): вблизи бьёт дубиной, издалека кидает снаряд
- ✅ Снаряды летят по дуге, разлетаются частицами при попадании в тайл
- ✅ Индикатор тревоги (!) над головой guard
- ✅ Автоматическая расстановка по сгенерированной карте (70% платформ)

### Пауэрапы
- ✅ Бутылка: +1 жизнь, пиксельный спрайт, покачивание
- ✅ Сигарета: ускорение 5 сек, анимированный дымок
- ✅ Автоматическая расстановка по карте
- ✅ HUD-сообщение в правом верхнем углу, полоска ускорения

### Архитектура
- ✅ Код разбит на 8 файлов: constants / map / player / enemies / items / particles / hud / game
- ✅ imageSmoothingEnabled = false везде + ResizeObserver
- ✅ findSpawnY — объекты всегда спавнятся над тайлом, никогда внутри
- ✅ tileHP — хранит повреждения тайлов
