# ЭТАП 1 — Анимация персонажа

## Правила (обязательно соблюдать)
- ВСЯ графика только через fillRect — никаких arc, font, drawImage
- Размер спрайта: 15×24 пикселей
- Каждый ряд спрайта: [x, y, width] — рисуется fillRect(ox+x, oy+y, width, 1)
- В покое персонаж стоит СПИНОЙ к камере
- При движении вправо — flip=false, влево — flip=true

## Спрайт STAY (стоит спиной, уже есть в game.js)
Использовать как базу для всех анимаций.

## Анимация idle (покачивание)
Не отдельный спрайт — просто смещаем STAY по Y:
```js
player.bobY = Math.sin(player.bobt * 1.5) * 0.8;
// применяем: py = Math.round(p.y + p.bobY)
```

## Логика смены кадров в update():
```js
// в объекте player добавить:
// frame: 0, ftick: 0, state: 'idle', bobt: 0, bobY: 0

p.ftick += dt;

if (!p.onGround) {
  p.state = 'jump';
} else if (dx !== 0) {
  p.state = 'run';
  if (p.ftick > 0.1) { p.frame = (p.frame + 1) % 4; p.ftick = 0; }
} else {
  p.state = 'idle';
  p.frame = 0;
  p.bobt += dt;
  p.bobY = Math.sin(p.bobt * 1.5) * 0.8;
}
```

## Логика выбора спрайта в draw():
```js
let spr = STAY;
if (p.state === 'jump') spr = JUMP;
else if (p.state === 'run') spr = p.frame % 2 === 0 ? RUN_0 : RUN_1;

const bobOffset = p.state === 'idle' ? Math.round(p.bobY) : 0;
drawSprite(spr, px, py + bobOffset, p.facing < 0);
```
