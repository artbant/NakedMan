# ROADMAP — Nuked Man
## Вертикальный срез: Локация 1

---

## ✅ СДЕЛАНО

### Персонаж
- Спрайт 15×24px попиксельно из SVG, стоит спиной
- Анимации: idle (покачивание), run (4 кадра), jump, punch (3 фазы)
- Физика: движение, прыжок, коллизия с тайлами
- Knockback при ударе — небольшой отскок назад
- Удар разрушает хрупкие тайлы перед собой

### Карта
- Процедурная генерация (42–60 тайлов, случайные высоты, пропасти 1–3 тайла)
- 8 типов текстур тайлов
- 3 типа тайлов по прочности: хрупкий (1 удар), средний (2 удара), прочный
- Визуальные маркеры разрушаемости + трещины при повреждении
- 3-слойный параллакс фон (всё через fillRect)
- Плавающие платформы всегда хрупкие

### Враги
- Тип A — патрульный: ходит туда-обратно, 2 HP, knockback
- Тип Б — сторожевой (guard): вблизи бьёт дубиной, издалека кидает снаряд, 1 HP
- Мигание при уроне, HP-полоска, взрыв пикселей при смерти
- Снаряды летят по дуге, разлетаются при попадании в тайл

### Пауэрапы
- Бутылка: +1 жизнь (макс 5), покачивается
- Сигарета: ускорение 5 сек + дымок
- HUD-сообщение при подборе, полоска ускорения

### Система жизней
- 3 жизни старт (макс 5)
- Смерть → respawn на том же уровне
- 0 жизней → Game Over → новый уровень

### Технические
- Файлы: constants / map / player / enemies / items / particles / hud / game
- imageSmoothingEnabled = false + ResizeObserver
- Пиксельный HUD через fillRect, никаких ctx.font
- findSpawnY — спавн всегда над тайлом

## ⬜ СЛЕДУЮЩИЕ ЭТАПЫ

### Этап 12 — Разрушаемые объекты
Ящики и бочки как отдельные объекты:
- Ящик (хрупкий) — 1 удар, много частиц
- Бочка — 2 удара, крупные куски
- Красная бочка — взрыв в радиусе, урон врагам
- Снаряды врагов тоже разрушают объекты

### Этап 13 — Финишная точка (портал)
- Пиксельный портал в конце уровня (мигающая арка)
- Касание → "YOU MADE IT" → новый уровень

### Этап 14 — Оружие для игрока
- Бутылка (3 удара), Дубина (5 ударов, 2 урона), Пистолет (6 патронов)
- HUD: иконка + заряды

### Этап 15 — Звук (Web Audio API)
- Прыжок, удар, смерть, подбор, разрушение тайла
- Простая генеративная фоновая музыка

### Этап 16 — Пауза и полировка
- ESC → пауза
- Плавный экран смерти и Game Over
- Счёт в HUD

### Этап 17 — Мобильное управление
- Альбомная ориентация + подсказка повернуть телефон
- Тест на реальном устройстве

### Этап 18 — Чекпоинты
- Пиксельный флажок посередине уровня
- Активируется при касании, respawn на нём

## Правила разработки
- ВСЯ графика через fillRect — никаких arc, font, drawImage
- imageSmoothingEnabled = false всегда
- Спавн объектов только через findSpawnY
- Пропасти не шире 3 тайлов
- Один файл = одна ответственность
